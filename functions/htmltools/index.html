    Generating HTML¶

    Tools for generating HTML snippets.

    emmet¶

    emmet(abbr) generated the HTML snippet according to abbreviation abbr in Emmet syntax.

    Danger

    Node.js and NPM must be properly installed and accessible on PATH.

    Warning

    When calling for the first time, proper Node.js dependencies will be installed, with the following message Installing Node.js dependencies... printed to standard error.

    Info

    This function is a wrapper of the official Node.js package @emmetio/expand-abbreviation.

    print(emmet("ul.nav>.nav-item{Item $}*2"))
# <ul class="nav">
#   <li class="nav-item">Item 1</li>
#   <li class="nav-item">Item 2</li>
# </ul>
